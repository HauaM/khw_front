# Suggested Commands

- `yarn install` (or `npm install`) — install dependencies defined in `package.json`.
- `cp .env.example .env` — copy the example env file and supply service credentials/URLs.
- `yarn dev` / `npm run dev` — run the Vite dev server (default `http://localhost:3000`).
- `yarn build` / `npm run build` — type-check and build the production bundle.
- `yarn preview` / `npm run preview` — spin up a local production preview of the built app.
- `yarn lint` — run ESLint (`--ext ts,tsx`, no warnings allowed) across the project.
- `git status`, `git diff`, `git add`, `git commit` — standard Git workflow on Linux (also available via `rg`/`ls` for searching/listing).
- `rg '<pattern>'` — fast ripgrep search for text in the repo.
- `node --version`, `yarn --version` (or `npm --version`) — confirm toolchain versions before working.
